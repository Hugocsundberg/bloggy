## Installation

1. #### Clone the repository

    `https://github.com/gildaeklof/bloggy`

2. #### Cd to the root of the project and run:

    `composer update`

3. #### Create a .env file with credentials to your database

4. #### Run:

    `php artisan migrate`

5. #### Run:

    `php artisan storage:link`

6. #### Run:

    `php artisan serve`
